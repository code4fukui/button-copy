# button-copy

One-click copy text from an input field.

## Demo
[https://code4fukui.github.io/button-copy/](https://code4fukui.github.io/button-copy/)

## Features
- Adds a "copy" button to an input field
- Copies the input text to the clipboard with one click

## Usage
Include the `button-copy.js` script in your HTML:

```html
<script type=module src=https://code4fukui.github.io/button-copy/button-copy.js></script>

<input id=inp value=test><button-copy target=inp>copy</button-copy>
```
